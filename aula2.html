Frontend Inteligente com JavaScript

🎯 Objetivo

Nesta aula, você vai evoluir o projeto aprendendo a trabalhar com múltiplos elementos e estados dinâmicos.

Também vai entender como o JavaScript pode controlar o comportamento visual da interface.

🧠 Revisão

O DOM representa a estrutura da página no navegador.

Com JavaScript, conseguimos acessar e modificar elementos.

Eventos permitem reagir às ações do usuário.

🚀 Manipulação de Classes

Podemos alterar o estilo de um elemento usando classes.


elemento.classList.add("ativa");
elemento.classList.remove("ativa");
elemento.classList.toggle("ativa");

O toggle alterna automaticamente o estado.

🧠 Estado Dinâmico

Agora não precisamos mais de variáveis separadas.

O próprio elemento indica seu estado.


if (elemento.classList.contains("ativa")) {
  // está ativo
}

📦 Trabalhando com Vários Elementos

Podemos selecionar vários elementos ao mesmo tempo:


const pecas = document.querySelectorAll(".peca");

Depois percorremos todos com:


pecas.forEach(function(peca) {
  // código aqui
});

🖱️ Eventos em Múltiplos Elementos

Cada peça pode reagir ao clique:


pecas.forEach(function(peca) {

  peca.addEventListener("click", function() {
    // ação ao clicar
  });

});

🧩 Evolução do Projeto

Agora o sistema fica mais avançado:

• várias peças funcionando ao mesmo tempo

• interação direta

• mudança visual dinâmica

💻 Exemplo Completo


// Seleciona todas as peças
const pecas = document.querySelectorAll(".peca");

// Seleciona contador
const contador = document.getElementById("contador");

// Função contador
function atualizar() {
  const ativas = document.querySelectorAll(".peca.ativa");
  contador.innerText = ativas.length;
}

// Evento
pecas.forEach(function(peca) {

  peca.addEventListener("click", function() {

    peca.classList.toggle("ativa");
    peca.classList.toggle("bloqueada");

    const icone = peca.querySelector(".icone");
    const texto = peca.querySelector("p");

    if (peca.classList.contains("ativa")) {
      icone.innerText = "🧩";
      texto.innerText = "Peça ativada";
    } else {
      icone.innerText = "🔒";
      texto.innerText = "Clique para ativar";
    }

    atualizar();

  });

});
