Introdução ao JavaScript e DOM

Objetivo

Nesta aula você vai entender como o JavaScript funciona no navegador e como ele interage com a página.

Também vai aprender a modificar elementos e responder às ações do usuário.

O que é o DOM?

O DOM é a representação da página que o navegador cria a partir do HTML.

Ele organiza tudo em uma estrutura que pode ser manipulada pelo JavaScript.

HTML, CSS e JavaScript

HTML define a estrutura.

CSS cuida do visual.

JavaScript adiciona comportamento.

Acessando elementos

Exemplo de acesso:


document.getElementById("meuElemento");

Eventos

Eventos são ações do usuário, como cliques:


elemento.addEventListener("click", function() {
  alert("Você clicou!");
});

Alterando conteúdo

Podemos mudar textos e estilos:


elemento.innerText = "Novo texto";
elemento.style.color = "red";

Classes e estados

Também é possível trabalhar com classes:


elemento.classList.add("ativa");
elemento.classList.remove("bloqueada");

Lógica

Controle com variáveis:


let ativo = false;

if (ativo === false) {
  // ação
}
